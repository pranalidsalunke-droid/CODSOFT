# Task 1: Number Guessing Game

This project is part of my Java Development Internship at CodSoft.

## Description
The program generates a random number within a specified range and allows the user to guess it.
The user receives feedback whether the guess is too high, too low, or correct.
The game has a limited number of attempts and supports multiple rounds.

## Concepts Used
- Java Basics
- Random class
- Scanner class
- Loops
- Conditional statements

## How to Run
1. Compile the program
2. Run the Java file
3. Enter guesses as prompted
